Enterprise Network Infrastructure Overhaul
Viion · Jul 2024 – Present
Networking
Problem

Legacy flat network with no segmentation causing broadcast storms, bottlenecks, and security gaps. Devices shared a single VLAN, making lateral movement trivial for any internal threat. Downtime was averaging 3–4 hours per week.

Solution

Redesigned the entire network topology using Layer 3 switches with inter-VLAN routing. Deployed Cisco enterprise routers and firewalls, configured OSPF for dynamic routing, implemented VLANs per department, and established a secure VPN gateway for remote staff.

Results
  • Network throughput increased by 75%
  • Weekly downtime reduced to near zero
  • Lateral movement threats eliminated via VLAN isolation
  • Remote access secured with split-tunnel VPN

Medical IT
Problem

Critical medical systems — including HIS (Hospital Information System), PACS imaging, and lab equipment — were running on unmanaged, unsecured infrastructure. Frequent outages disrupted patient care workflows. No disaster recovery plan existed.

  • Unauthorised access incidents dropped by 90%
  • Recovery time objective (RTO) reduced to under 2 hours
  • Full VLAN isolation for all medical devices

Endpoint Security & Group Policy Hardening
Viion · Multi-Site Deployment
Security
Problem

Workstations across multiple office locations had inconsistent security configurations, outdated OS patches, and no centralised policy enforcement. Shadow IT and unauthorised USB usage were common, creating significant data exfiltration risk.

Solution

Rolled out a comprehensive Group Policy Object (GPO) baseline across all domain-joined machines — disabling USB storage, enforcing password complexity, enabling BitLocker encryption, and locking down local admin rights. Deployed WSUS for automated Windows Update management across all sites.

Results
  • 100% policy compliance across all endpoints
  • USB-based incidents eliminated completely
  • Patch compliance improved from ~40% to 98%
  • Zero privilege escalation incidents post-deployment

Virtualisation
Problem

Physical servers running Windows Server 2008 R2 were approaching end-of-support, running on ageing hardware with no redundancy. Single points of failure were risking business continuity. Expanding physical infrastructure was cost-prohibitive.

Designed and executed a P2V (Physical-to-Virtual) migration plan using Microsoft Hyper-V and VMware. Migrated 6 physical servers to virtual machines on a high-availability cluster, upgraded all VMs to Windows Server 2019, and configured live migration for zero-downtime maintenance windows.

Cisco Infrastructure Revamp
Main Sharia Hospital · Aug 2024
Networking
Problem

A sprawling flat network was causing massive broadcast storms, leading to daily intermittent connectivity drops and IP conflicts across departments. Frequent network loops were crippling core medical applications.

Solution

Redesigned the topology using a hierarchical distribution model. Deployed Cisco L3 core switches, routing between cleanly segmented VLANs via OSPF. Implemented Rapid Spanning Tree Protocol (RSTP) to eliminate loop risks intrinsically.

Results
  • Broadcast storms eradicated; isolated securely to logical departments
  • Network downtime reduced by 99%
  • Improved inter-department routing efficiency
  • Future-proofed OSPF base mapping established for future scalability
Cisco IOSOSPFVLAN TrunkingRSTPL3 Switching

Healthcare Network Segmentation & Security
Al Salam Medical Centre · Oct 2024
Security
Problem

Patients, guests, and administrative staff were connecting to the same unsegmented network. This open ecosystem posed severe security and compliance risks to unencrypted patient records and medical imaging devices (PACS).

Solution

Deployed and configured Fortinet/Sophos Perimeter Firewalls. Restructured wireless infrastructure enforcing WPA3 Enterprise for doctors and a restricted Captive Portal for patients isolated entirely onto a separate internet-only VLAN.

Results
  • Guests effectively firewalled from clinical databases (100% Policy Compliance)
  • Traffic shaping prioritized medical data streams over guest bandwidth
  • Eliminated unauthenticated access to the physical network ports
  • Intrusion/Malware blocking via Unified Threat Management (UTM)
